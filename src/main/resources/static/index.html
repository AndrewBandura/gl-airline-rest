Airline company REST API

You can process CRUD operations with airline company. Aircraft cannot exist without parent airline. So you can operate aircraft only through parent airline.
Type URL Info Input JSON Output JSON
GET /api/company get all companies with all nested aircraft
GET /api/company/{id} get company by ID with nested aircraft
POST /api/company create company with nested aircraft
PUT /api/company/{id} update company with nested aircraft. Also is used to create new aircraft
DELETE /api/company/{id} delete company
GET /api/company/{id}/capacity calculate total capacity of all aircraft in the airline
GET /api/company/{id}/carrying-capacity calculate total carrying capacity of all aircraft in the airline
GET /api/company/{id}/aircraft/sort get aircraft of the airline sorted by flight range
GET /api/company/{id}/aircraft/consumption/{from}/{to} get aircraft of the airline filtered by fuel consumption
